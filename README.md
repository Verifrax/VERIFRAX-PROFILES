# VERIFRAX PROFILES

DETERMINISTIC POLICY PROFILES FOR VERIFRAX VERIFICATION.

PROFILES DEFINE:
- REQUIRED FIELDS
- ACCEPTANCE THRESHOLDS
- DOMAIN CONSTRAINTS

PROFILES DO NOT CHANGE THE SPEC.
